# Budimas Stock Transfer

### Frontend Modul Budimas Stock Transfer

#### How To Install ?
 
1. Clone this repo
```cmd
git clone https://gitlab.com/fahmi6326/budimas-stock-transfer-2.git
```

2. open the project
```cmd
cd budimas-stock-transfer-2
code .
```

3. install the packages
```cmd
npm install
```

4. run the project
```cmd
npm run serve
```
