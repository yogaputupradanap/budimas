# Budimas Distribusi

### Frontend Modul Budimas Distribusi

#### How To Install ?

1. Clone this repo
```cmd
git clone https://gitlab.com/fahmi6326/budimas-distribusi.git
```

2. open the project
```cmd
cd budimas-distribusi
code .
```

3. install the packages
```cmd
npm install
```

4. run the project
```cmd
npm run serve
```
