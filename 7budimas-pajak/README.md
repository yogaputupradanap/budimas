# Budimas Piutang

### Frontend Modul Budimas Piutang

#### How To Install ?

1. Clone this repo
```cmd
git clone https://gitlab.com/fahmi6326/budimas-piutang.git
```

2. open the project
```cmd
cd budimas-piutang
code .
```

3. install the packages
```cmd
npm install
```

4. run the project
```cmd
npm run serve
```
