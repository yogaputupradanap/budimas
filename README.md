# budimas

untuk menjalankan python
1. pip install -r requirements.txt
2. jika ada yang error silahkan pip install yang diperlukan
3. jika sudah selesai atur database sesuai dengan yang ada di server
4. db terlampir
5. untuk start python di dev "set FLASK_ENV=production && python app.py"
6. jika source laravel pakai composer install atau update
7. jika source vue pakai npm install
8. kedua nya untuk install node modulnya
9. untuk source selain python pakai npm start serve kecuali laravel bisa langunsg pakai link di laragon
10. semua di jalankan lewat terminal selain source laravel
